HTML Element의 display 속성 2가지 : Block / Inline

block

div element는 block element. 항상 새로운 줄부터 시작하며, 전체 가로 폭(width)을 100% 차지함.

paragraph element도 block element.

inline

a element는 inline element. 새로운 줄(new line)부터 시작하지 않고, 자신이 필요한 영역만큼만 차지함.

p element 내부에 중첩(nesting)된 span element는 자신의 영역만 차지함.

입력 form : 도 마찬가지.
